# LOOM | Product Context

## Vision Statement
LOOM is a premium QR code platform that transforms static physical links into dynamic, trackable, and brandable digital bridges for creators and teams.

## Feature/Problem Matrix
| Problem | Feature | Benefit |
| :--- | :--- | :--- |
| Static codes are "dead" once printed. | Dynamic Redirects | Update URLs after printing; no reprinting costs. |
| Basic QR codes look generic. | Design Studio | Premium branding (logos, HSL colors, eye styles). |
| No visibility into offline engagement. | Scan Analytics | Real-time tracking of location, device, and frequency. |
| Fragmented tools for QR management. | Centralized Library | Manage, search, and edit all codes in one dashboard. |

## Success Metrics (KPIs)
1. **Conversion**: >20% of free users upgrade to Pro within 90 days.
2. **Reliability**: QR Redirect latency < 150ms globally.
3. **Engagement**: Average of 5 scans per active dynamic QR code.
